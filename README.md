**Grocery Shop Seller Simulation (Python)**

This project is a simple command-line grocery shop billing system written in Python.
It simulates the interaction between a seller and customers in a grocery store, allowing the seller to record purchased items, calculate the total bill, and generate a receipt.
The program repeatedly accepts customers, processes their shopping details, and prints a final receipt with the customer name, total amount, and date/time of purchase.

**Features**
1. Interactive seller–customer conversation simulation
2. Input for customer name, quantity, and item price
3. Automatic total bill calculation
4. Option to continue shopping or generate receipt
5. Displays purchase date and day using Python datetime
6. Supports multiple customers in a loop
7. Includes small real-time delays using the time module for realistic interaction

**Technologies Used**
1. Python
2. datetime module (for receipt date and time)
3. time module (to simulate delays)

**How It Works**
1. The seller asks the customer if shopping is complete.
2. The customer provides purchased item details.
3. The program calculates the total amount.
4. The seller asks if the customer wants to continue shopping.
5. If not, the system prints a receipt with purchase details and timestamp.
6. The seller can proceed to the next customer.
